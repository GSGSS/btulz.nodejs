# btulz.nodejs
这是Node.js环境下的开发工具集。

### 安装说明 | install
* nodejs环境下：
~~~
npm install -g btulz
~~~

### 使用说明 | instructions
#### types-*，解析声明库
* 解析openui5库文件：
~~~
btulz types-ui5 
~~~
